# Save Memory

Immediately persist the current session context to memory.

## Process

Update `index.md` with current session context (active explorations, skill level changes, project context). Checkpoint `progress.md` with any new terms explored or quiz results. Update `patterns.md` if new preferences were discovered. Append to `chronology.md` if a significant milestone occurred.

## Output

Confirm save with brief summary: "Memory saved. {brief-summary-of-what-was-updated}"
